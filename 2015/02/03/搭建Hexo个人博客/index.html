0. 什么是Hexo

Hexo是一个功能强大的博客框架，主要使用Markdown格式来书写博客，拥有大量漂亮的主题可供选择，并且生成静态文件可供快速访问。


官方文档链接：http://hexo.io/docs/index.html

1. 安装

需求

安装Hexo前，你需要先安装Git及Node.js

Git

  • Windows: 可以下载msysgit
  • Mac：可以通过Homebrw安装
  • Linux：可通过命令安装 sudo apt-get/yum install git-core

Node.js

先安装nvm

再用nvm安装Node.js
